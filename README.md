EJOY 2D
=======

Make
====

For Windows , you need mingw32 .

* Install Lua 5.2.3
* Install glew 1.9
* make or make mingw
* ej2d examples/ex01.lua to test

For Linux ,

* Install Lua 5.2.3
* Install glew 1.9
* Install freetype 2
* make or make linux
* ./ej2d examples/ex01.lua to test
